# WhatsApp Web Socket

A node application connect to WhatsApp web use socket

## Installation

Tested on : 
- node `16.16.0`
- npm `8.11.10`

Run
```bash
npm install
```
or (to install based on `package-lock.json`) 
```bash
npm ci
```
